# 📊 Otchet po Meta Bot — Инструкция

## Что делает этот бот
- Каждую ночь в случайное время (23:00–23:59 по Ташкенту) отправляет отчёт в Telegram группу клиента
- Берёт данные напрямую из Facebook Ads API
- Показывает: бюджет, лиды, звонки, клики, охват

---

## Установка на Railway (пошагово)

### Шаг 1 — Подготовка файлов
Скачай эти файлы: `index.js` и `package.json`

### Шаг 2 — GitHub
1. Зайди на github.com
2. Нажми "New repository" → назови `otchet-bot`
3. Загрузи оба файла
4. Нажми "Commit changes"

### Шаг 3 — Railway
1. Зайди на railway.app
2. Нажми "New Project" → "Deploy from GitHub repo"
3. Выбери репозиторий `otchet-bot`
4. Railway сам установит зависимости и запустит

### Шаг 4 — Переменные окружения (Variables)
В Railway нажми на проект → Variables → добавь:

| Переменная | Значение |
|---|---|
| BOT_TOKEN | твой токен от @BotFather |
| FB_ACCESS_TOKEN | токен из Facebook Business Manager |

### Шаг 5 — Получить FB_ACCESS_TOKEN
1. Зайди на developers.facebook.com
2. Создай приложение типа "Business"
3. Добавь продукт "Marketing API"
4. Сгенерируй долгосрочный токен (Long-lived token)

## Добавление нового клиента

В файле `index.js` найди раздел `КЛИЕНТЫ` и добавь:

```javascript
{
  name: 'Название компании',
  groupChatId: '-100XXXXXXXXXX',  // ID группы (узнать командой /id)
  fbAdAccountId: 'act_XXXXXXXXX', // ID рекламного аккаунта Facebook
},
```

### Как узнать ID группы:
1. Добавь бота в группу клиента
2. Напиши в группе `/id`
3. Бот ответит ID группы

### Как узнать ID рекламного аккаунта Facebook:
1. Зайди в Facebook Ads Manager
2. В адресной строке будет: `act_XXXXXXXXX` — это и есть ID

## Команды бота

| Команда | Что делает |
|---|---|
| /start | Приветствие и ID чата |
| /id | Показывает ID группы |
| /test | Отправить тестовый отчёт прямо сейчас |
| /status | Список всех клиентов |
